Naira - Virtual AI Assistant

Naira is a web-based AI assistant that supports both voice and text interaction. Featuring a futuristic UI with a glowing core animation, it enables real-time, intelligent conversations using Mistral-7B-Instruct via OpenRouter.


Technologies Used:
Frontend: HTML, CSS, JavaScript

Backend & Hosting: Node.js (Cloudflare), Netlify
Version Control: GitHub
